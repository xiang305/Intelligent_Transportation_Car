# Iot_物流運輸概念車
## 1.關於專案
這是一個可以用在物流運輸方面的遙控車，主要用途為使用者能觀看實時畫面操控車子移動，車子可記錄兩點之間的路線
## 2.所需材料
- [雙層四驅自走車底盤](https://www.tenlong.com.tw/products/10241289256) * 1
- Rasberry Pi 4 * 1
- 麵包板 * 1
- L298N馬達驅動模組 * 1
- 1.5v電池 * 4
- 四顆並聯電池盒(包含正負極單芯線) * 1
- 穩定5V的行動電源(可用樹莓派UPS 鋰電池擴充板替代) * 1
- 數條公對公、公對母、母對母杜邦線
- 捆線帶(非必要，用於整理杜邦線)
- 雙面膠(非必要，用於固定車上裝置)
